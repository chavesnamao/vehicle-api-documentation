# Versão 0.0.3 (15-09-2025)

## Sobre

* <strong>fix:</strong> Campo ```acceptExchange: boolean``` incluído na documentação

Prezados,

Gostaríamos de informar que, embora o campo **acceptExchange** já estivesse disponível para envio na API há cerca de 2 anos, ele ainda não constava em nossa documentação oficial. 

A partir desta versão, a documentação foi atualizada para refletir corretamente o suporte a este campo, que pode ser utilizado para indicar se o veículo permite ou não trocas.

Recomendamos que todos os desenvolvedores revisem a documentação atualizada para garantir o uso correto deste campo [clicando aqui](../02-veiculos/vehicle-object.md).

Atenciosamente.
